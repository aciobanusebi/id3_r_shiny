The structure of the file must be the following:
  • first line: types of columns separated by (a) white space(s) [d - discrete; c - continuous]
  • second line: column names separated by (a) white space(s)
  • the other lines: an instance on each line.
The output column is always the first one.
Example of file: 
d c
AreAritmie Varsta
0 40
1 60
1 62
